# AegisStream AI Service

FastAPI-based intelligence service for AegisStream Studio.

## Phase 5 scope

This service currently provides:

- service skeleton
- typed request/response contracts
- config and environment support
- startup/shutdown lifecycle
- logging
- error handling
- stub endpoints for:
  - document ingest
  - evidence retrieval
  - grounded Q&A
  - triage
  - eval runs
